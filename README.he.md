# Generate Video — סקיל ליצירת וידאו ב-Claude Code

סקיל מודרך וצעד-אחר-צעד ל-Claude Code שמייצר סרטוני AI קולנועיים באמצעות **HiggsField MCP**. הסקיל שואל אתכם ארבע שאלות, כותב תסריט קולנועי לאישורכם, ואז מייצר את הסרטון — ומחזיר קישור הורדה ישיר.

---

## תכונות עיקריות

- קלט מודרך שלב אחר שלב (ז'אנר, נושא, תמונת רפרנס, מודל)
- יצירת תסריט קולנועי ברמת במאי מקצועי
- אישור תסריט לפני כל יצירה
- העלאת תמונות רפרנס ושמירה על זהות דמויות דרך HiggsField MCP
- מעקב אוטומטי עד שהסרטון מוכן
- קישור הורדה ישיר ל-MP4 בסיום
- תמיכה במספר מודלים: Seedance 2.0 ,Kling 3.0 ,Veo 3, Grok Video

---

## דרישות

- [Claude Code](https://claude.ai/code) — CLI או אפליקציית שולחן עבודה
- HiggsField MCP מוגדר בהגדרות Claude Code
- חשבון HiggsField פעיל עם קרדיטים

---

## התקנה

העתיקו את `SKILL.md` לתיקיית הסקילים של הפרויקט:

```bash
mkdir -p .claude/skills/generate-video
cp SKILL.md .claude/skills/generate-video/SKILL.md
```

הפעילו מחדש את Claude Code — הסקיל יופיע אוטומטית.

---

## שימוש

הקלידו `/generate-video` ב-Claude Code. הסקיל ישאל:

1. **ז'אנר** — לדוגמה: `אקשן`, `דרמה`, `מדע בדיוני`, `אימה`, `אפי`, `נואר`
2. **נושא / רעיון** — תארו את הסצנה או הסיפור
3. **תמונת רפרנס** — נתיב לקובץ או `none` אם אין
4. **מודל** — ברירת מחדל: Seedance 2.0 (הטוב ביותר לשמירת זהות דמויות)

לאחר איסוף התשובות, הסקיל מציג תסריט קולנועי לאישורכם. לאחר אישור, הסרטון מיוצר דרך HiggsField וקישור ההורדה מוחזר אוטומטית.
